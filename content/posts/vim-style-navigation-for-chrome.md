The vimium extension for Chrome is a dream come true for me. So many useful shortcuts. After install press "?" for a list of shortcuts. "o" for instance opens the menu/searchbar, 'j' to scroll up, 'k' to scroll down, 'J' for next tab, 'K' for previous tab, 'x' to close the current tab, and more.
[Vimium Github](https://github.com/philc/vimium)
